🚀 How to Deploy ### Option 1 — Netlify Drop (fastest, ~30 seconds) 1. Rename `cs_knowledge_base.html` → `index.html` 2. Go to [netlify.com/drop](https://netlify.com/drop) 3. Drag and drop the file 4. Get an instant live URL like `https://random-name.netlify.app` ### Option 2 — GitHub Pages (free, permanent URL) 1. Create a new GitHub repository 2. Upload `cs_knowledge_base.html` and rename it `index.html` 3. Go to **Settings → Pages → Source: main branch** 4. Your URL: `https://yourusername.github.io/repo-name` ### Option 3 — Vercel 1. Go to [vercel.com](https://vercel.com) 2. Import from GitHub or drop the file 3.
